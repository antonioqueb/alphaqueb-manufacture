# Ajuste de legibilidad

Se aumentó el tamaño y contraste de microtextos, metadatos, labels de KPI, numeradores, tags, notas de metodología y textos auxiliares.

Objetivo: evitar que textos técnicos o secundarios se vean demasiado pequeños, delgados u opacos en pantallas de baja calidad.

Archivos afectados principalmente:

- `app/globals.css`
- `components/HomeLanding.jsx`
- `components/ContactSection.jsx`

Los cambios se hicieron como override global para sostener consistencia en todo el sitio.
